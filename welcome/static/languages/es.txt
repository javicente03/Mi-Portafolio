work1-title=>Sistema de gestión administrativo para Corpotulipa (Trabajo Especial de Grado) 
work1-def=>Trabajo especial de grado como requisito para optar al título de Ingeniero de Sistemas. Sistema desarrollado en el lenguaje Php conectado a una base de datos MySQL, dirigido a las oficinas de Recursos Humanos y Administración y Finanzas de CORPOTULIPA (Falcón, Venezuela), consta de 3 módulos (Caja chica, Préstamo de bienes muebles, Solicitudes de permiso y adiestramientos). Para el frontend se optó por usar el framework Materialize CSS, debido a su gala de herramientas para un diseño intuitivo y agradable, así como JavaScript Vanilla.
work2-title=>Panel de integración (Amazon - Mercado Libre)
work2-def=>Sistema de gestión para cuentas de mercado libre realizado en Django, consumiendo el API proporcionado por Mercado Libre y llevando a cabo un scraping a Amazon, este software permite a sus usuarios publicar articulos en masa a su cuenta de ML, configurando todo lo relacionado a su producto (precio, categoria, atributos) con suma facilidad. Igualmente el sistema posee la cualidad de responder preguntas realizadas en las publicaciones en seguimiento de manera automática.
